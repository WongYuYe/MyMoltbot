MyMoltbot - Todo List 小程序

注意：这是一个微信小程序项目，需要在微信开发者工具中打开使用。

项目说明

这是一个基于微信原生框架开发的待办事项（Todo List）小程序应用。

要体验完整功能，请：

  1. 下载并安装微信开发者工具
  2. 克隆或下载此项目代码
  3. 在微信开发者工具中导入项目
  4. 编译运行体验完整功能

主要功能

✓ 添加待办事项

快速添加新的待办事项，支持回车快捷键

✓ 完成状态切换

点击复选框标记完成/未完成状态

✓ 删除功能

支持删除不需要的待办事项

✓ 数据持久化

使用本地存储保存数据，重启不丢失

✓ 实时统计

显示总数量、待完成、已完成统计

✓ 响应式设计

支持深色模式，界面美观大方
