# MyMoltbot - 微信小程序 Todo List 应用

这是一个基于微信原生框架开发的待办事项（Todo List）小程序应用。

## 功能特性

- 添加新的待办事项
- 标记完成/未完成状态
- 删除待办事项
- 实时统计（总数量、待完成、已完成）
- 本地数据持久化
- 响应式设计，支持深色模式

## 技术栈

- 微信小程序原生框架 (WXML, WXSS, JavaScript)
- 小程序云存储实现数据持久化

## 快速开始

1. 下载并安装微信开发者工具
2. 克隆此项目
3. 在微信开发者工具中导入项目
4. 编译运行

## 功能说明

- 在顶部输入框中输入待办事项内容
- 点击"添加"按钮或按回车键添加待办事项
- 点击复选框标记完成/未完成状态
- 点击"删除"按钮移除待办事项
- 底部实时显示统计信息
